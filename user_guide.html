Image Multi Cropper User Guide

Welcome to the Image Multi Cropper tool! This guide will help you understand how to use the application.

Loading an Image

  1. Click the "Load Image" button or select "Load Image" from the File menu.
  2. Browse to the image file you want to crop and select it.

Creating Rectangles

  1. Toggle the draw mode by clicking the "Toggle Draw Mode" button.
  2. Click and drag on the image to create a rectangle.
  3. Release the mouse button to complete the rectangle and enter a name for the crop.

Resizing Rectangles

  1. Hover the mouse cursor near the edge or corner of an existing rectangle.
  2. When the cursor changes to a resize icon, click and drag to resize the rectangle.

Saving Crops

  1. Click the "Save Crops" button or select "Save Crops" from the File menu.
  2. All defined rectangles will be saved as separate image files in a "cropped_images" directory within the current working directory.

Additional Features

  • Use the rectangle list panel on the right to view and manage the created rectangles.
  • Select a rectangle from the list to highlight it on the image.
